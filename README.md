# nvimipdb
a simple and minimalistic ipdb (ipython debugger) for neovim

## Prerequisites
* neovim
* ipdb (install for example by pip install ipdb)
* (later on neomake)

## Functions 
* Ipdb
* ReplSendPyLine
* ReplSendPyFile
* ReplSendPySelecton
* ...

## Key maps
* <Plug>GoToDebugLine
* ...
